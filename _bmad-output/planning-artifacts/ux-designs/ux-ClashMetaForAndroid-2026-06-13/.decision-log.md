# UX Decision Log
*Started: 2026-06-13*

## Context
User has requested a UX overhaul of the Connections page. Current page is functional but feels rough, ugly, and incomplete.

## User Feedback
- Visual identity approved: Modern dark-mode card style.
- Information Architecture: User selected "Accordion / 手风琴折叠式" (Option 1).
- Clarification 1: User pointed out the missing expanded state UI.
- Clarification 2: User requested that the cards should be more compact ("卡片不要这么高") and there should be no vertical line on the left side of child items ("左边不要有竖线").

## Decisions Made
- Visual identity: Modern dark-mode card style (Material 3).
- Navigation/IA: Accordion expand/collapse for grouped app connections.
- Sub-item Styling: Compact card heights. No vertical tree-lines for children. Rely purely on indentation or subtle background differences for hierarchy.

## Open Questions
- None. Ready for Implementation.
